# SQL Joins
A complete SQL practice project covering table creation, data insertion, updates, deletions, filtering, aggregation, and joins. It helps understand how data connects across tables and how queries return meaningful combined results.

## Features
- Clear relationship between Customers and Orders.
- Includes basic join types.
- Eaasy to run in MySQL environment.

## Technologies Used
- MySQL
- MySQL Workbench

## Join Queries
- Inner Join - Fetches only matching rows
- Left Join - Fetches all customers and related orders
- Right Join - Fetches all orders and related customers
- Full Join - Combines left and right join results using UNION

## Steps to run
- Open MySQL Workbench.
- Create a database "joins_db".
- Create the Customers and Orders table.
- Insert the data into both tables.
- Run each join query to observe the result difference.
- Use SELECT * to verify the table data anytime.
